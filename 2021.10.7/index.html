以下是笔者的学习计划

# 前端学习计划

# 一、学习总览

首先学习三大件 html css js 并掌握牢固，三大框架 vue react Augular 掌握两种，至此完成基础学习。

# 二、学习资源

目前计划利用 mooc，b 站，菜鸟教程，w3school，github 等网站资源进行学习

  • 三大件学习：1. b 站 pink 老师 html+css 2. b 站 pink 老师 JavaScript

  • 框架学习：1. vue 2. React

  • 学习概览：1. web 开发

  • 学习路线总览

# 三、未来展望

# 一、生涯规划

目前主要打算在大四之前完成基础前端开发的学习，日后工作以前端为基础，逐步向后端转移，在按部就班的学习中应补充计算机理论基础知识，例如数据结构与算法，设计模式，计算机组成原理，数据库原理，计算机网络等课程。复习 c 语言，Python，学习 JAVA,kotlin 等语言为安卓开发打基础，学有余力情况下学习 c++，复习单片机等知识，为日后可能进入嵌入式行业做基础。
